CS143 Grammar Solver ProjectThis is the grammar solver assignment that I did in CSE 143. It works by using grammar rules recorded in Backus-Naur form, and comparing those rules to the current sentence. Each word is chosen in the sentence based on the word type in the grammar. I'm enjoying my CS143 class because I'm learning a lot about how computers use very simple data structures in order to build up to very complex problems. In this case, it surprised me how simple it was to programmatically construct complex sentences.
